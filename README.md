# Mill Game – Deluxe Variant (2022)

This is a multiplayer version of the classic **Nine Men's Morris** game, developed in Java using Swing and custom socket-based communication.

🔐 **Note:** Login and authentication logic has been removed from this public version for security and privacy reasons.

## 🎮 Features
- Multiplayer game logic and board rules
- Java Swing-based GUI
- Game state managed by client-server structure (server code excluded)

## 🧰 How to Run
- Use a Java IDE (like NetBeans or IntelliJ)
- Compile and run `MainWindowNew.java` in the client
- The public version is GUI-only (no login or server connection)

## 📁 Project Structure
- `MillFinal` - Client-side UI and game logic
- `MillServerFinal` - (Removed: used for authentication and server logic)
- `resources/` - Images and assets
